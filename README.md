# Параметрический алгоритм синусоиды

Скрипт `SineWaveDrawer` в Unity процедурно строит синусоиду и перемещает точку вдоль неё.  
Форма полностью зависит от параметров.

## Принцип работы

Скрипт:

1. Рисует синусоиду через `LineRenderer`
2. Перемещает объект (`movingPoint`) вдоль кривой

Пересчёт происходит каждый кадр в `Update()`.

---

## Основные параметры

- `amplitude` — высота волны  
- `wavelength` — расстояние между пиками  
- `length` — общая длина волны  
- `points` — плавность линии  
- `normalizedPosition (0–1)` — положение точки вдоль кривой  

---

## Построение волны

Для каждой точки вычисляются координаты:

```
t = i / (points - 1)
x = t * length
y = sin(x * 2π / wavelength) * amplitude
```

Изменение параметров сразу меняет форму:

| Параметр | Результат |
|----------|-----------|
| Amplitude | Волна выше или ниже |
| Wavelength | Волна чаще или реже |
| Length | Волна длиннее |
| Points | Более плавная линия |
| Normalized Position | Движение точки |

# Параметрическая модель «Стол» и «Стул»

Модели работают по тому же принципу: изменяем параметры — геометрия перестраивается автоматически.

---

# Стол

## Основные параметры

- `height` (740–760 мм)
- `width`
- `depth`
- `tabletopThickness`

Высота ножек:

```
legHeight = height - tabletopThickness
```

При изменении высоты:

- столешница поднимается
- ножки автоматически удлиняются

Если включён `ergonomicMode`, высота ограничивается диапазоном 740–760 мм.

Изменение ширины и глубины раздвигает ножки по углам.

# Стул

## Основные параметры

- `seatHeight` (400–450 мм)
- `seatWidth`
- `seatDepth`
- `backrestHeight`
- `backrestAngle`

Высота ножек:

```
legHeight = seatHeight - seatThickness
```

При изменении высоты сиденья:

- сиденье поднимается
- ножки удлиняются автоматически

Спинка масштабируется по ширине сиденья и наклоняется по углу.  
Можно включить удлинение задних ножек (`autoLegLength`).

# Связь стола и стула

Комфортная разница:

```
высота стола − высота сиденья = 270–320 мм
```

Слишком маленькая разница — тесно.  
Слишком большая — неудобно.

# Итог

Во всех примерах используется один принцип:

- меняем параметры
- Unity автоматически пересчитывает геометрию
- объект перестраивается без ручного редактирования

Базовая реализация процедурной и параметрической модели.
